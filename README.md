# programming_blocksite

## サービス概要
YoutubeやTwitterなどのwebサービスが大好きすぎて、プログラミング学習が手に付かない人に<br>プログラミングの問題をクリアすることで、webサービスの利用ができるようになる仕組みを提供する<br>webサービスをご褒美としたプログラミング学習サービスです。

## 登場人物
プログラミング学習しているのに、ついつい動画やSNSなどのwebサービスを使ってしまう人

## ユーザーが抱える課題
プログラミングを勉強しないといけないのに、動画やSNSを見て時間を無駄にしてしまう。<br>
(僕は1日10時間以上、動画やSNSを見てしまうことがある)<br>
プログラミングを勉強していない罪悪感を感じながら動画やSNSを見ているが、本当はプログラミングの勉強をしっかりやってから、動画やSNSを楽しみたい。

## 解決方法
あらかじめ利用制限したいwebサービスのサイトURLを指定する。制限したURLに飛ぶと、Paizaのようなプログラミング問題が出題される。問題をクリアするとサイトが利用できるようになる。

## プロダクト
プログラミングの問題をクリアすることで、webサービスの利用ができるようにするサービス

## マーケット
10~40代のプログラミングができる人
